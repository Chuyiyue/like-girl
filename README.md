#### 使用说明

- 文章书写需了解HTML标签基本语法
- 可以百度查询资料学习
- 部署后会有一篇默认文章 可以参考书写格式

#### 食用方法
1.打开根目录下的`admin`文件夹
2.接着找到`Config_DB.php`文件打开编辑你的数据库相关信息
3.请认真填写安全码 尽量设置的`复杂难以猜测` 修改密码等敏感信息需输入安全码
4.最后到数据库导入sql文件（`like_girl.sql`）
5.默认账号密码：`admin`/`love`
